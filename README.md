# [ 0 ] Null

> *Less bloat. More speed. Pure UI.*

`Null` — это ультра-минималистичный, легковесный кроссплатформенный графический клиент для прямого взаимодействия с LLM-моделями через API OpenRouter. Разработан на Python с использованием современных асинхронных потоков, как производительная и стабильная альтернатива тяжелым GUI-клиентам на базе Electron (таким как Jan).

---

## ⚡️ Ключевые преимущества

* **Ультра-минимализм (Zero Bloat):** Никакого Chromium под капотом. Приложение потребляет в десятки раз меньше оперативной памяти по сравнению с аналогами, обеспечивая максимальную отзывчивость интерфейса.
* **Прямой стриминг (Direct Stream):** Ответы от серверов (включая 120B гигантов вроде `NVIDIA Nemotron`) выводятся на экран в ту же миллисекунду, когда генерируются, без задержек на рендеринг тяжелой графики.
* **Идеальный контроль контекста:** Никакого скрытого накопления мусорного кэша и логов, забивающих систему и вызывающих ошибки таймаута. Чистая, контролируемая работа с историей сообщений.
* **Стильный GUI-дизайн:** Монохромный, аккуратный интерфейс с нативной поддержкой темной темы, вдохновленный эстетикой Nothing и ориентированный на продуктивную работу разработчиков.
