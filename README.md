# [ 0 ] Null

> *Less bloat. More speed. Pure code.*

`Null` — это ультра-минималистичный, легковесный консольный терминал на Python для прямого взаимодействия с LLM-моделями через API OpenRouter. Создан как производительная альтернатива тяжелым GUI-клиентам на базе Electron.

---

## ⚡️ Почему Null?

* **Zero Bloat:** 0% лишнего графического мусора, 0% Chromium под капотом. Программа расходует ~10 МБ оперативной памяти.
* **Direct Stream:** Ответы от серверов (включая 120B гигантов вроде `NVIDIA Nemotron`) выводятся на экран в ту же миллисекунду, когда генерируются.
* **Идеальный контроль:** Никакого скрытого кэша истории, забивающего память. Контекст очищается простым перезапуском.
